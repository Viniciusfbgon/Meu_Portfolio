Trainee DataMLOps - Desafios e Entregas

Durante este projeto, enfrentei 5 sprints desafiadoras que me prepararam para atuar com infraestrutura, automação e práticas modernas de deploy. Ao longo da jornada, desenvolvi habilidades práticas fundamentais em DevOps e Cloud Computing:

• Sprint 1: Versionamento com Git e GitLab, comandos avançados em Linux, automação de tarefas com crontab e introdução ao Google Cloud Platform (GCP).

• Sprint 2: Criação e gerenciamento de containers com Docker, manipulação de buckets no GCP e primeiros conceitos de cultura DevOps.

• Sprint 3: Fundamentos de redes de computadores, aplicação de metodologias ágeis com Scrum e construção de pipelines automatizados com GitLab CI/CD.

• Sprint 4: Deploy completo de uma API Node.js com banco PostgreSQL utilizando Docker Compose na GCP, com foco em escalabilidade e resiliência.

• Sprint 5: Implementação de CI/CD com ambientes isolados (build, test, deploy) e automação de backups diários para o Cloud Storage.

Python + Mediapipe

Primeiro projeto em Python com Inteligência Artificial utilizando Mediapipe. Este sistema realiza o mapeamento da mão em tempo real por meio da biblioteca Mediapipe, que detecta e rastreia landmarks específicos da palma e dos dedos. A partir da análise da posição relativa desses pontos, o algoritmo identifica individualmente os dedos estendidos e realiza a contagem automática da quantidade de dedos visíveis na imagem.

O sistema responde dinamicamente à movimentação da mão, atualizando a contagem conforme a configuração dos dedos muda, por exemplo, ao levantar três dedos, a saída exibida é “3”; ao levantar cinco, a saída muda para “5”.
